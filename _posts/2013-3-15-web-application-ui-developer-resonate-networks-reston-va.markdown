We are currently seeking a Web Application / UI Developer to join our growing team.  For the position, previous experience developing scalable, rich, internet application user interfaces is a must. The individual must have the desire and ability to learn quickly.  The environment here is fast paced with changes and challenges on a daily basis, and so the ability to keep up with this environment is a necessity. The candidate will be responsible for implementation of the platform user interface, and will work closely with the product manager and data visualization designer.

So if you enjoy the challenge of a fast-paced start-up environment and like to break innovative new ground, we want to talk to you!

##Key Responsibilities

* UX/UI implementation
* Develop & maintain front end code as well as some mid-tier code
* Implement new, ground-breaking interaction and visualization interfaces
* Develop UI that provides incredible rich user experience
* Work closely and effectively with the middle tier and database development team

##Qualifications & Experience Requirements

* 2-8 years experience in launching successful interactive web applications
* Superior written and verbal communication skills
* Experience in advertising network, digital media agency, or other advertising related experience a plus
* Solid experience in developing rich internet applications
* Adept with web standards and cross-browser development
* Expert at UI/UX Web Design
* Data Visualization/Dashboard Design experience a plus

###Skills

* Expert skills with JavaScript, AJAX, HTML and CSS technologies
* Prior experience building Web Application User Interfaces
* Experience with MVC or MVP preferred
* Experience working with various Web API’s (GoogleMaps, GoogleChart, Twitter, etc.)
* Flash/Flex expertise a strong plus
* Google Web Toolkit (GWT) experience a strong plus

###Educational Requirements

* Bachelor’s Degree required
